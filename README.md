# 📡 Telecom Customer Intelligence: Segmentation & Churn Prediction

## Project Overview
This project implements an **Intelligent Business System** designed to reduce customer attrition in the telecom industry. It combines **Unsupervised Learning** (to understand customer types) and **Supervised Learning** (to predict future behavior).

## Features
- **Data Preprocessing:** Automated handling of missing values and categorical encoding.
- **Clustering (Unsupervised):** K-Means algorithm to segment customers into 3 behavioral groups.
- **Prediction (Supervised):** Random Forest Classifier to predict the probability of a customer leaving (Churn).
- **Interactive Interface:** A live dashboard built with Streamlit for real-time predictions.

## Project Structure
- `research.ipynb`: Complete data analysis, math formulas, and model evaluation.
- `app.py`: The production-ready Streamlit application.
- `requirements.txt`: List of necessary Python libraries.
